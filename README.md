# Simple GraphQL server

Built with Node.js, Express, Apollo, GraphQL, Sequelize

## Installation

- create `.env` file with Postgres DB connection variables:

```
DATABASE=
DATABASE_USER=
DATABASE_PASSWORD=
```

- `npm install`
- `npm start`
